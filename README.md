# AI 레시피 추천 서비스 My Little Chef

- 냉장고 재료, 인원수, 식사 유형을 입력하면 AI가 맞춤 레시피 3가지를 추천해주는 웹 서비스
- Claude를 활용한 개발 전반에 걸친 코드 작성 및 디버깅
- Groq API(LLaMA 모델)를 활용한 한국 가정식 레시피 자동 생성
- Naver 이미지 검색 API 연동으로 레시피에 맞는 음식 이미지 자동 표시 및 출처 기재
- Next.js API Routes를 활용한 서버 기능 구현 (REST API)
- MongoDB Atlas를 활용한 레시피 북마크 저장 및 불러오기
- React 기반 컴포넌트 구조로 UI를 분리하여 재사용성과 유지보수성 향상
- TypeScript를 적용하여 안정적인 타입 관리
- Tailwind CSS를 사용한 반응형 UI 디자인 (모바일/태블릿/PC 대응)
